🧬 UK NEQAS H&I Scheme 3 Bookmarklet

This tool helps you populate and extract HLA antibody form data on the migrated 2026 UK NEQAS H&I Scheme 3 results form.

Note: This version targets the current Power Apps based form. The retired hidden-field form from the old site is no longer supported.

🔧 How to Use

  1. Drag the button below to your bookmarks bar.
  2. Open the migrated UK NEQAS H&I Scheme 3 results form.
  3. Click the bookmarklet. Upload and Download buttons will appear.

⬇️ Drag this to your bookmarks bar:

🧬 NEQAS H&I Scheme 3 Tool

📄 Upload a CSV File

The uploaded CSV should have the format:

HLA,MFI,UNACCEPTABLE
A2,24000,1
B44,2500,0
DR1,30000,1
DQA1*01:01,26000,0

HLA can be a Class I, Class II, DQA, or DPA row label, for example A2, DR1, DPB1*04:01, DQA1*01:01, or 01:01 for DQA/DPA rows.

MFI can be a plain number such as 24000. Re-uploading downloaded ranges such as "2,000-4,999" is also supported.

UNACCEPTABLE should be 1 to tick the field or 0 to leave it unticked.

NB: The 2026 page builds Class I, Class II, DQA, and DPA grid sections in the page DOM, so one CSV can include rows from all four sections.

⬇️ Download as CSV File

After processing, you can download a CSV containing checked rows:

HLA,MFI,UNACCEPTABLE
A2,"20,000-24,999",1
B44,"2,000-4,999",0
DR1,"≥25,000",1

The downloaded file will be automatically named from the sample code in the form title when available. Otherwise it uses hla_output.csv.
